‚öôÔ∏è How It Works

  1. Capture ‚Äî Takes a screenshot of your desktop (or reads wallpaper file)
  2. Divide ‚Äî Splits the image into a grid of square tiles
  3. Scramble ‚Äî Randomly flips each tile horizontally or vertically
  4. Animate ‚Äî Each tile flips back with a 3D effect, starting from the center
  5. Reveal ‚Äî Cut lines fade away to show the seamless original image
  6. Repeat ‚Äî Optionally auto-restarts after a delay

üëÅÔ∏è Eyes Effect

The eyes effect detects or lets you select eye regions, then animates the pupils to look around naturally.

How It Works

  • Pupil Movement ‚Äî Only the inner pupil/iris area moves (center 70%)
  • Static Edges ‚Äî The outer edges of the eye stay completely fixed
  • Smooth Blending ‚Äî Seamless transition between moving and static areas
  • Natural Motion ‚Äî Sine-wave based movement for realistic gaze patterns

Eye Selection Methods

üñ±Ô∏è First Time - Select & Save

python3 animate.py -c -e eyes --select-eyes

Click to draw polygons around each eye. Saved to eyes.cfg

üîÑ Subsequent Runs - Use Saved

python3 animate.py -c -e eyes

Loads saved eye regions from ~/.config/.../eyes.cfg

üìê Manual Coordinates

python3 animate.py -c -e eyes --eye-pos '120,120,100,70;330,96,120,80'

Specify exact x,y,width,height for each eye

ü´• Hidden Effect

The hidden effect removes objects from the image using OpenCV's inpainting algorithm. Perfect for making watches, jewelry, bracelets, and other accessories invisible.

How It Works

  • Selection ‚Äî Draw polygons around objects you want to remove
  • Inpainting ‚Äî OpenCV fills the selected areas with surrounding content
  • Reveal ‚Äî Smooth crossfade from original to modified image

Usage

üñ±Ô∏è First Time - Select & Save

Draw polygons around objects. Saved to hidden.cfg

üîÑ Subsequent Runs - Use Saved

Loads saved regions from ~/.config/.../hidden.cfg

üîÑ Clone Effect

The clone effect replaces selected areas with content from another region you specify. Perfect for replacing hair with skin, or filling areas with specific textures.

How It Works

  • Target Selection ‚Äî First, draw a polygon around what you want to HIDE (shown in RED)
  • Source Selection ‚Äî Then, draw a polygon around what you want to CLONE FROM (shown in GREEN)
  • Cloning ‚Äî The source content is scaled and copied to the target area

Usage

üñ±Ô∏è First Time - Select & Save

Select target (red), then source (green). Saved to hidden.cfg

üîÑ Subsequent Runs - Use Saved

Loads saved regions from ~/.config/.../hidden.cfg

üíæ Saved Configurations

Hidden and Clone selections are saved to ~/.config/desktop_animation_tool/hidden.cfg. Reuse them by running without --select-objects!

üîß Troubleshooting

Black screen with -c flag

Make sure scrot is installed:

sudo apt install scrot

Wrong wallpaper detected

Use the -c flag to capture actual desktop instead of reading wallpaper file.

Desktop mode not working

  • Only works on X11, not Wayland
  • Install required tools: sudo apt install x11-utils xdotool

Animation too fast/slow

Adjust with --speed (lower = slower, higher = faster)

Eyes not detected

  • Use --select-eyes for interactive polygon selection
  • Or use --eye-pos to manually specify coordinates
  • Make sure OpenCV is installed: pip install opencv-python --break-system-packages
